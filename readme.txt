== Description ==

Better Thumbnail Sizes adds an option in the Media settings to place all image sizes in a sub-directory of the uploads directory.

If the plugin is used as a Must-Use plugin, the Media settings option will be hidden and the functionality automatically enabled.

== Installation ==

= Manual installation =

Download the Better Thumbnail Sizes plugin and uploading it to your server via your favorite FTP application. The WordPress codex contains [instructions on how to do this here](https://codex.wordpress.org/Managing_Plugins#Manual_Plugin_Installation).
